# Analisis Sentimen - AB

Project ini dibuat untuk memenuhi penilaian tipe A pada mata kuliah terkait analisis sentimen. Sistem ini bertujuan untuk melakukan klasifikasi sentimen pada data teks menggunakan algoritma **K-Nearest Neighbor (KNN)** dengan bahasa pemrograman **PHP**.

## Deskripsi Project

Analisis Sentimen - AB adalah project yang berfokus pada proses analisis sentimen terhadap data teks. Analisis sentimen digunakan untuk mengetahui kecenderungan opini dari suatu teks, apakah termasuk sentimen positif, negatif, atau netral.

Project ini dirancang menggunakan bahasa pemrograman **PHP** dengan menerapkan algoritma **K-Nearest Neighbor (KNN)** sebagai metode klasifikasi. Algoritma KNN bekerja dengan membandingkan data teks yang diuji dengan data latih yang sudah memiliki label sentimen, kemudian menentukan hasil klasifikasi berdasarkan kedekatan data tersebut.
